# Versions

Use this page to navigate documentation versions.

| Version       | Status   | Docs                                       |
| ------------- | -------- | ------------------------------------------ |
| v1.1 (latest) | Stable   | [Open latest](/advanced/tab-item)          |
| v1.0.0        | Archived | [Open v1.0.0](/v1.0.0/components/tab-item) |

## Version Policy

- `latest` always points to the newest stable docs.
- Older versions remain available for maintenance and audits.
- New breaking changes should create a new version folder under `pages/`.
